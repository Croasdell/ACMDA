# Handyman Plus Van Website

This is the Flask-based website for Handyman Plus Van, featuring:
- Home page
- Services page
- Booking form (integrated with SQLite)
- Contact page

## Running Locally

1. Install dependencies:
   ```
   pip install flask
   ```
2. Ensure you have the database set up (run the booking system once to create `acmda.db`).
3. Start the server:
   ```
   python app.py
   ```
4. Visit [http://localhost:5000](http://localhost:5000) in your browser.
